# 🔍 AI写作痕迹检测清单

## 一、词汇层痕迹

### 1.1 高频连接词堆砌
AI最爱用的"三件套"：不仅...而且...同时...此外...另外...
- 检测方法：同一段落出现3个以上并列连接词
- 严重程度：⭐⭐⭐⭐⭐

### 1.2 模板化开头
- "在当今社会..." / "随着XX的快速发展..."
- "近年来，XX领域蓬勃发展..."
- "XX是一个备受关注的话题..."
- 严重程度：⭐⭐⭐⭐⭐

### 1.3 空洞修饰词
- "具有重要意义" / "发挥着重要作用"
- "不可忽视的" / "至关重要的"
- "显著的" / "深远的"
- 检测方法：删掉这些词后意思是否变化？不变就是废话
- 严重程度：⭐⭐⭐⭐

### 1.4 模糊限定词
- "一定程度上" / "某种程度上"
- "相关" / "相关方面"
- "某些" / "一些"
- 严重程度：⭐⭐⭐

### 1.5 假大词
- "赋能" / "抓手" / "闭环" / "颗粒度" / "底层逻辑"
- "深度学习" / "全面升级" / "系统性优化"
- 严重程度：⭐⭐⭐

## 二、句式层痕迹

### 2.1 "总-分"段落结构
每段开头是概括句，后面跟2-3个展开点，AI的"八股文"。
- 检测方法：如果每段首句都能当小标题，大概率是AI
- 严重程度：⭐⭐⭐⭐⭐

### 2.2 并列句式过度使用
- "首先...其次...再次...最后..."
- "一方面...另一方面..."
- "第一...第二...第三..."
- 正常人写东西不会这么工整
- 严重程度：⭐⭐⭐⭐

### 2.3 过度使用被动语态
- "被认为..." / "被广泛应用于..."
- "得到了显著提升"
- 严重程度：⭐⭐⭐

### 2.4 总结句模板
- "综上所述..." / "总而言之..."
- "总的来说..." / "不难发现..."
- 严重程度：⭐⭐⭐⭐

### 2.5 长句无断句
单句超过40字没有标点断开，AI生成的典型特征。
- 严重程度：⭐⭐⭐

## 三、结构层痕迹

### 3.1 过于工整的结构
全文结构像论文：引言→正文1→正文2→正文3→结论
- 正常人的写作会有跳跃、插入、补充
- 严重程度：⭐⭐⭐⭐

### 3.2 段落长度高度一致
每段都是3-5行，段段均匀，像流水线产品。
- 严重程度：⭐⭐⭐

### 3.3 缺乏过渡的生硬转折
段落之间用"此外""同时"连接，但内容并不递进。
- 严重程度：⭐⭐⭐

### 3.4 结尾必有"升华"
每篇文章结尾都要拔高立意、喊口号。
- 严重程度：⭐⭐⭐⭐

## 四、内容层痕迹

### 4.1 缺乏具体细节
- "很多人" → 具体多少？哪些人？
- "相关研究表明" → 哪个研究？谁做的？
- "某公司" → 哪个公司？
- 严重程度：⭐⭐⭐⭐⭐

### 4.2 缺乏个人视角
全文没有"我"字，全程第三人称客观叙述。
- 正常人分享经验会自然带入"我"
- 严重程度：⭐⭐⭐⭐⭐

### 4.3 观点中立到无聊
每个话题都是"有优点也有缺点"，不敢表态。
- 严重程度：⭐⭐⭐⭐

### 4.4 例子泛泛而谈
"例如，某公司通过XX实现了XX" → 没有具体公司名、具体数据
- 严重程度：⭐⭐⭐⭐

### 4.5 情感扁平
全文语气一致，没有情绪波动、没有吐槽、没有惊喜。
- 严重程度：⭐⭐⭐

## 五、综合评分标准

| AI痕迹评分 | 特征 | 处理建议 |
|-----------|------|---------|
| 1-2分 | 几乎无AI痕迹 | 轻度润色即可 |
| 3-4分 | 轻微AI痕迹 | 针对性修改问题点 |
| 5-6分 | 明显AI痕迹 | 需要结构性改写 |
| 7-8分 | 重度AI痕迹 | 需要全面重写 |
| 9-10分 | 典型AI范文 | 需要从骨架开始重构 |

## 六、快速检测口诀

> **开头看"随着"，中间看"不仅"，结尾看"总之"。**
> **段段皆总分，句句都并列，篇篇无"我"字。**
> **——十有八九是AI。**
