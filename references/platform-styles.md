# 📱 平台风格适配指南

## 小红书 (Xiaohongshu / RED)

### 平台调性
- 用户画像：18-35岁女性为主，追求品质生活、种草拔草
- 内容调性：真诚分享、闺蜜聊天感、实用干货

### 风格规则
- **标题**：15字以内，用emoji开头，制造好奇心（"天呐！这个XX也太好用了吧"）
- **开头**：直接说痛点或惊喜，不要铺垫（"姐妹们！我终于找到了！"）
- **emoji使用**：每2-3行一个emoji，用在句首或句尾点缀，不要堆砌
  - 常用：✨💕🔥👏🏻😍🤔💡📌‼️
  - 分类标记：✅❌⭐🔵
- **段落**：短段落（2-4行），大量换行，阅读零压力
- **编号**：干货用数字编号（1️⃣2️⃣3️⃣或❶❷❸）
- **语气**：亲切、有温度、像朋友聊天
  - ✅ "我真的会谢！" "谁懂啊！" "绝绝子！"
  - ❌ "笔者认为" "经研究发现"
- **结尾**：互动引导（"你们觉得呢？" "评论区告诉我~" "点赞收藏不迷路💕"）
- **标签**：3-5个话题标签，放在末尾（#好物分享 #生活小技巧）

### 格式模板
```
[emoji] 标题

[emoji] 开头：痛点/惊喜引入（1-2句）

[emoji] 正文要点1
具体内容...

[emoji] 正文要点2
具体内容...

[emoji] 正文要点3
具体内容~

[emoji] 总结/互动引导

#标签1 #标签2 #标签3
```

## 抖音 (Douyin / TikTok)

### 平台调性
- 用户画像：全年龄段，追求娱乐、快节奏、即时满足
- 内容调性：口语化、有节奏感、强钩子、适合口播

### 风格规则
- **前3秒钩子**：必须在前两句抓住注意力
  - "你知道吗？90%的人都不知道..."
  - "千万别再这样做了！"
  - "我后悔没早点知道..."
  - "三个字：绝了。"
- **句式**：短句为主，一句话不超过15个字
- **节奏**：快节奏推进，信息密度高，不废话
- **口语化程度**：最高级别
  - 可以用："我跟你说" "真的假的" "就离谱" "家人们谁懂"
  - 避免：长从句、书面语、学术表达
- **互动设计**：口播互动（"你猜怎么着？" "等等，还没完"）
- **结尾**：行动号召（"赶紧试试" "点赞收藏" "评论区见"）
- **标签**：2-3个热门话题标签

### 口播节奏模板
```
[钩子：3秒内抓住注意力]
[冲突/反转：制造期待]
[核心信息：分点快速输出]
[行动号召：引导互动]
```

## 知乎 (Zhihu)

### 平台调性
- 用户画像：25-40岁，高学历，追求深度思考和专业分析
- 内容调性：理性、有深度、有论据、个人见解

### 风格规则
- **开头**：直接回应问题，亮明观点（"先说结论：..." "这个问题我太有发言权了"）
- **结构**：逻辑清晰，层层递进
  - 可用：问题拆解 → 分析论证 → 案例佐证 → 个人观点
- **论证**：需要数据/案例/经历支撑观点
  - ✅ "根据XX数据..." "我之前在XX公司时..."
  - ❌ "众所周知" "毋庸置疑"
- **专业度**：保持一定专业性，但不要掉书袋
- **个人色彩**：知乎鼓励"过来人"视角
  - "说个真事" "踩过坑的人告诉你" "作为XX从业者"
- **emoji**：极少使用，最多在小标题用符号标记（→、·、◆）
- **段落**：中等长度，每段一个论点，3-6行
- **结尾**：个人总结/观点升华，不要鸡汤（"以上是我的看法，欢迎讨论"）

### 格式模板
```
先说结论：[一句话观点]

[论点1]
[分析 + 案例/数据]

[论点2]
[分析 + 案例/数据]

[个人经历/见解]

[总结观点 + 开放讨论]
```

## 微信公众号 (WeChat Official Account)

### 平台调性
- 用户画像：25-45岁，职场人士为主，阅读场景偏深度
- 内容调性：有质感、有观点、排版清晰

### 风格规则
- **标题**：信息量大，制造阅读欲望（不超过32字）
  - 数字型："3个方法，让我的工作效率提升了200%"
  - 悬念型："被裁员后，我发现了一个残酷真相"
  - 干货型："2024年最值得关注的XX趋势"
- **开头**：场景代入/故事切入，前3行决定是否继续阅读
- **结构**：小标题分段，每段5-8行，逻辑清晰
- **emoji**：适度使用，主要在小标题和重点标注处
- **排版意识**：
  - 重要句子单独成段（金句感）
  - 适当使用引用块、加粗、分割线
  - 段落间留白
- **语气**：介于专业和亲切之间，"有温度的理性"
- **结尾**：金句收尾 + 引导关注/转发

## 微博 (Weibo)

### 平台调性
- 用户画像：全年龄段，追求热点、即时资讯、社交互动
- 内容调性：短平快、有态度、易传播

### 风格规则
- **字数**：140字以内为佳，长微博另说
- **开头**：直接说观点或抛出话题
- **语气**：有态度、有情绪、敢表达
- **话题**：绑定1-2个热搜/话题标签
- **互动**：以提问或投票结尾
- **emoji**：1-3个点缀即可

## 朋友圈 (WeChat Moments)

### 平台调性
- 私人社交，熟人可见
- 真实、不做作、有生活气息

### 风格规则
- **字数**：1-3行为主
- **语气**：随意、真实、像说话
- **emoji**：1-2个，自然融入
- **避免**：营销感、鸡汤、转发语
- **场景**：分享日常、记录感悟、晒图配文

## 通用避坑清单

### 所有平台通用的AI味特征（必须消除）
1. "在当今社会/随着科技的发展" → 直接说事
2. "不仅...而且...同时...此外..." → 拆成短句
3. "综上所述/总而言之" → 用金句或反问收尾
4. "值得注意的是/需要指出的是" → 直接说
5. "具有重要意义/发挥着重要作用" → 说具体有什么用
6. "相关/相关方面/相关部门" → 说清楚具体是谁
7. "一定程度上/某种程度上" → 要么说清楚程度，要么删掉
8. 全文没有"我" → 必须有个人视角
9. 每段都是"总-分"结构 → 变换结构
10. 结尾必有鸡汤总结 → 用态度或互动替代
