# 🧪 AI内容炼金术师 (Content Alchemist)

> 将AI味十足的文本炼成有温度、有个性、平台适配的优质内容

## 🎯 解决什么问题？

2025-2026年，AI写作已经渗透到每个人的日常——用ChatGPT写文案、用Claude写报告、用各种AI工具生成内容。但一个普遍的痛点是：**AI写出来的东西，一眼就能看出是AI写的**。

"在当今社会..."、"不仅...而且..."、"综上所述..."——这些AI味十足的表达，不仅让读者反感，还会被平台算法限流（小红书、抖音等平台已明确打击AI生成内容）。

**Content Alchemist 就是解决这个问题的。**

## ✨ 核心能力

| 能力 | 说明 |
|------|------|
| 🔍 AI痕迹检测 | 精准识别文本中的AI写作模式，给出1-10分评分 |
| 🧬 去AI化改写 | 消除模板化表达，注入个人视角和情感温度 |
| 📱 平台风格适配 | 一键适配小红书/抖音/知乎/公众号/微博/朋友圈 |
| 🌡️ 情感温度调节 | 在专业↔亲切、严肃↔幽默之间灵活切换 |
| ✅ 质量保障 | 确保改写后信息完整、准确、无错误 |

## 🔄 工作流程

```
输入文本 → 分析诊断 → 炼金改写 → 质量校验 → 输出交付
   │          │          │          │          │
   │     AI痕迹评分    结构重塑     信息完整性    改写报告
   │     问题定位      语言人味化    自然度校验    改写内容
   │     需求确认      情感注入     平台合规      延伸建议
   │                  平台适配
```

## 🏆 创新亮点

1. **赛道首创**：首个专注于"去AI化"的内容优化工具，填补市场空白
2. **平台感知**：不是简单改写，而是深度理解各平台调性和算法偏好
3. **诊断先行**：先量化AI痕迹评分，再针对性优化，而非暴力重写
4. **保留精髓**：在去除AI味的同时，确保核心信息零丢失
5. **多场景覆盖**：从种草文案到专业分析，从口播脚本到朋友圈配文

## 📋 技术规范

- **输入**：任意中文文本 + 可选参数（平台、风格、字数）
- **输出**：改写后文本 + 改写报告 + 延伸建议
- **鲁棒性**：支持短文本（50字+）到长文本（5000字+），异常输入不崩溃
- **安全性**：不收集用户数据，不存储输入内容，纯文本处理无安全风险
